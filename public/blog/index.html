Using camera modules on a Raspberry Pi running Ubuntu 25.04 (Plucky Puffin)

With the release of Ubuntu 25.04 Plucky Puffin, the camera stack now works on a Raspberry Pi running Ubuntu. The main changes were adding the new PiSP driver to libcamera and adding the rpicam-apps and picamera2 userland packages (along with their dependencies). We also tried to support IMX500’s (Raspberry Pi AI camera) AI features (except the Hailo preprocessing models), but that comes with a catch described later. We have made the usage of the camera stack as close to Raspberry Pi OS as possible, so you can use your existing setup and scripts and run them on Ubuntu as is.
